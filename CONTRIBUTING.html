# Contributing Guides

首先报名成为志愿者，目前在招募译者。

# 翻译步骤

  1. 认领翻译文章：创建 Issue，认领待翻译的文章。
  2. 开始翻译：根据 Issue 创建文章的分支，开始翻译。翻译的时候建议一段原文一段中文。记得多提交。
  3. 完成后发起 Pull Request，等待审阅。

# 审阅步骤

  1. 查看 Pull Request，对比翻译结果。
  2. 如有修改请标记。
  3. 确认质量合格后，合并 Pull Request 到 master 分支。
